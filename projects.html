Human Activity Recognition Using Persistent Homology

Human Activity Recognition (HAR) plays a big part in many things like health, living in a smart way, and working out. While old ways of doing machine learning and deep learning are good at HAR, they don't do well when there is a lot of noisy and hard to work with data. In our work, we show a new way to do this that uses Persistent Homology.

Parkinson's Disease Detection Using AI In Healthcare With MATLAB

Parkinson's Disease (PD) is a neurological disease that presents with difficulty in the early diagnosis. The aim of this project is to identify PD using AI approaches in MATLAB. Support Vector Machine or SVM is applied for developing a robust classification model. Clinically and biomedically,vocal patterns and motility scores are considered. The model has details of the aims to forecast whether or not a patient is a Parkinson's Disease patient.
